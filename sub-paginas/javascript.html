O que eu aprendi?
Eu aprendi Javascript básico através do site FreeCodeCamp, e a experiência foi incrível. Fiz todo o curso de 300 horas de Javascript, que é totalmente prático, e além disso, concluí os 4 projetos que são necessários para obter a certificação. O curso me ensinou os conceitos fundamentais da linguagem de programação, como variáveis, funções, loops e condicionais, e me deu a oportunidade de praticar esses conceitos em exercícios e projetos reais. A plataforma é fácil de usar, com aulas bem estruturadas e desafios divertidos que me mantiveram motivado a continuar aprendendo. Agora, me sinto confiante para usar Javascript em meus próprios projetos e estou animado para continuar minha jornada de aprendizado com o FreeCodeCamp.

Eu criei um "caderno de Javascript" que é um repositório no GitHub onde treinei e estudei a linguagem antes de começar os projetos práticos no FreeCodeCamp. O caderno contém diversos códigos que eu escrevi para praticar os conceitos que aprendi, além de anotações sobre as funções, variáveis e outros tópicos importantes de JavaScript. Essas anotações me ajudaram a revisar os conceitos e a entender melhor a linguagem. O GitHub é uma ótima plataforma para compartilhar projetos e trabalhar em equipe, e meu caderno de JavaScript foi uma forma de armazenar meu progresso e compartilhar meu conhecimento com outros desenvolvedores. Depois de criar o caderno e praticar bastante, me senti mais preparado para os desafios práticos do FreeCodeCamp.
